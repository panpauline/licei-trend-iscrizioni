## Come muoverti, in breve

```
                    ┌─────────────────────────────────┐
                    │   La mia app dei bandi           │
                    └─────────────────────────────────┘
                                  │
        ┌─────────────────────────┼─────────────────────────┐
        │                         │                         │
        ▼                         ▼                         ▼
   🔔 ISSUES               📚 risultati.md            ▶️ ACTIONS
  bandi nuovi              archivio storico          lancia ricerca
  (+ email)                (tutto lo storico)        manuale subito
```

**Ogni giorno ti basta:**
- controllare la **mail** (arriva solo se c'è qualcosa di nuovo), oppure
- aprire il link **🔔 Issues** per vedere le ultime segnalazioni.

Gli altri due link servono per rivedere tutto lo storico (📚) o per forzare
una ricerca immediata (▶️).
